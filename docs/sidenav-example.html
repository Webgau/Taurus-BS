Sidebar Navigation - Dokumentation

Diese Seite demonstriert die Sidebar-Navigation aus dem FeuerwehrManager-Projekt.

Enthaltene Dateien:
  • sidenav.css - Alle CSS-Styles
  • sidenav.js - JavaScript-Funktionalität
  • sidenav-example.html - Diese Beispieldatei
Features:
  • Responsive Design (Mobile/Desktop)
  • Dark Mode / Light Mode Support
  • Collapsible Submenus (bis zu 3 Ebenen)
  • Profile-Dropdown
  • Sidebar-Suche/Filter
  • Minify-Modus (nur Icons)
  • Scroll-Memory (merkt Scroll-Position)
  • CSS-Variablen für einfache Anpassung
Abhängigkeiten:
  • Bootstrap 5.x
  • Font Awesome 6.x
CSS-Klassen
Klasse Beschreibung
.app-sidebar Haupt-Sidebar-Container
.menu Menu-Container
.menu-item Einzelnes Menu-Element
.menu-link Menu-Link
.menu-submenu Submenu-Container
.has-sub Item mit Submenu
.active Aktives Item
.menu-header Abschnitts-Header
.menu-divider Trennlinie
Data-Attribute
Attribut Funktion
data-toggle="app-sidebar-mobile" Mobile Toggle-Button
data-dismiss="app-sidebar-mobile" Mobile Sidebar schließen
data-toggle="app-sidebar-minify" Minify-Button
data-toggle="app-sidebar-profile" Profil-Dropdown Toggle
data-sidebar-search="true" Such-Input aktivieren
data-bs-theme="dark" Dark Mode aktivieren
Theme-Anpassung

Die Sidebar verwendet CSS-Variablen, die leicht angepasst werden können. Wechsle zwischen Light und Dark Mode mit dem Button unten rechts. 

/* Beispiel: Eigene Farben definieren */
:root {
    --bs-app-sidebar-bg: #1a1a2e;
    --bs-app-sidebar-profile-bg: #16213e;
    --bs-app-sidebar-component-active-bg: #0f3460;
    --bs-app-theme: #e94560;
}
